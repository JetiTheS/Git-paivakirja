# Oppimispäiväkirja: Paikallinen git

__Mikä osion tehtävissä oli vaikeaa ja mikä helppoa? Mikä auttoi minua oppimaan? Miten selvitin esteet?__

Git:in käyttö on yleisellä tasolla jo aika tuttua kun on sitä muilla kursseilla jo käyttänyt, ehkä paluu edelliseen committiin toi päänvaivaa kun ei sitä ole tarvinnut käyttää enkä meinannun millään saada terminaalia hyväksymään revert komennon committia. Git:in käyttö on aika suoraviivaista kunhan muistaa/ymmärtää mikä käsky tekee mitäkin.
Ohjeet olivat kyllä kattavat, ainoastaan revert komennon kanssa oli ihmeteltävää. Oikeilla hakusanoilla google osasi näyttää oikean toimintatavan.

## Osiossa käyttämäni Git-komennot

| Komento | Kuvaus |
| --------| ------ |
| status | Näyttää missä tilassa git tallennus on |
| add | Lisää muutokset/lisäykset talletettavaksi  |
| commit | Vie muutokset version hallintaan |
| log | Näyttää commitit ja niiden kommentit |
| switch | Vaihtaa haaraa |
| reset | peruuttaa add komennon |
| restore | palauttaa tiedot tuoreimpaan versioon |
| revert | palauttaa versiohallinassa olevat tiedot edelliseen committiin |

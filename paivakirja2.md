# Oppimispäiväkirja: Hajautettu git

__Mikä osion tehtävissä oli vaikeaa ja mikä helppoa? Mikä auttoi minua oppimaan? Miten selvitin esteet, jotka vaikuttivat tehtävän suorittamiseen?__

Git:iin siirtäminen sujui ongelmitta, git antaa hyvät ohjeet miten edetä repositorion pystyttämiseen ja liittämiseen. 

## Osiossa käyttämäni Git-komennot

| Komento | Kuvaus |
| --------| ------ |
| remote add origin (url) | yhdistää gittiin perustetun repositorion ja oman repon |
| pull | vetää kaiken tiedon mitä etäreposta löytyy ja yhdistää omaan |
| push | lähettää kaikki tehdyt muutokset etärepoon |
| fetch | hakee kaiken tiedon etäreposta mutta ei yhditä omaan vaan on erillinen tietonsa |
| merge | yhdistää kaksi haaraa ja luo uuden tallennuksen |
